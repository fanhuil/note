添加文件到暂存区

```shell
git add (.|filename)
```

将暂存区文件提交到代码仓库

```shell
git commit -m "commit info"
```

比较不同版本的文件

```shell
git diff HEAD -- filename4
```

撤销提交暂存区文件

```shell
git reset HEAD filename
```

回退版本

```bash
git reset --hard HEAD^ 回到上一个版本，一个^回退一个版本
git reset --hard HEAD~n 回退n个版本
git reset --hard 版本id 跳到某个版本
```
